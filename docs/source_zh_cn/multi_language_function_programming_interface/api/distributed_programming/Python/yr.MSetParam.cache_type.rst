yr.MSetParam.cache_type
------------------------------------

.. warning::

   ``yr.MSetParam`` 已废弃，仅为兼容历史版本保留。新代码请勿继续使用。

.. py:attribute:: MSetParam.cache_type
   :type: CacheType
   :value: 0

   标识分配的存储介质类型（内存或磁盘）。
   可选参数为 ``CacheType.Memory`` （内存介质）和 ``CacheType.Disk`` （磁盘介质），默认值为 ``CacheType.Memory``。
